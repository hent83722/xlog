# Zyrnix v1.0.4 - Quick Feature Summary

## Three Major Features

### 1. ðŸ“„ Configuration File Support
Load logger configurations from JSON files without recompiling.

**Quick Example:**
```cpp
Zyrnix::ConfigLoader::load_from_json("config.json");
auto loggers = Zyrnix::ConfigLoader::create_loggers();
loggers["app"]->info("Loaded from config!");
```

**Config Example (config.json):**
```json
{
  "loggers": [
    {
      "name": "app",
      "level": "info",
      "async": true,
      "sinks": [
        {"type": "stdout"},
        {"type": "file", "path": "app.log"},
        {"type": "rotating", "path": "app.log", "max_size": 10485760, "max_files": 5}
      ]
    }
  ]
}
```

**Benefits:**
- Change log levels without recompiling
- Different configs for dev/staging/production
- Easy A/B testing

---

### 2. ðŸš¨ Signal-Safe Logging
Crash-safe logging for signal handlers.

**Quick Example:**
```cpp
#include <Zyrnix/sinks/signal_safe_sink.hpp>

auto crash_sink = std::make_shared<Zyrnix::SignalSafeSink>("crash.log");
auto logger = std::make_shared<Zyrnix::Logger>("crash");
logger->add_sink(crash_sink);

void crash_handler(int sig) {
    logger->log(Zyrnix::LogLevel::Critical, "Caught signal!");
    crash_sink->flush();
    _exit(1);
}

signal(SIGSEGV, crash_handler);
```

**Features:**
- Async-signal-safe (only POSIX safe functions)
- Lock-free ring buffer
- No malloc/free
- Reliable crash logging

---

### 3. ðŸ“¦ Conditional Compilation
Reduce binary size by 50-70KB.

**Quick Example:**
```bash
# Minimal build (smallest)
cmake -DXLOG_MINIMAL=ON ..

# Custom build
cmake -DXLOG_ENABLE_ASYNC=OFF -DXLOG_ENABLE_JSON=OFF ..

# Compile flags
g++ -DXLOG_NO_ASYNC -DXLOG_NO_JSON main.cpp -lZyrnix
```

**Feature Flags:**
- `XLOG_NO_ASYNC` - Disable async logging (~15-20KB)
- `XLOG_NO_JSON` - Disable JSON logging (~10-15KB)
- `XLOG_NO_NETWORK` - Disable network sinks (~8-12KB)
- `XLOG_NO_COLORS` - Disable colors (~2-3KB)
- `XLOG_NO_FILE_ROTATION` - Disable rotation (~5-8KB)
- `XLOG_NO_CONTEXT` - Disable contexts (~3-5KB)
- `XLOG_NO_FILTERS` - Disable filters (~2-4KB)
- `XLOG_MINIMAL` - Disable all (~50-70KB)

**Feature Detection:**
```cpp
#if XLOG_HAS_ASYNC
    auto logger = Zyrnix::Logger::create_async("app");
#else
    auto logger = Zyrnix::Logger::create_stdout_logger("app");
#endif
```

## Migration from v1.0.3

**No breaking changes!** All existing code works without modification.

New features are opt-in:
1. Use config files if you want dynamic configuration
2. Add signal-safe sink if you need crash logging
3. Use feature flags if you need smaller binaries

## Installation

```bash
git clone https://github.com/hent83722/Zyrnix.git
cd Zyrnix
git checkout v1.0.4
bash scripts/build.sh && sudo bash scripts/install.sh
```

Or with CMake:
```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --parallel
sudo cmake --install .
```
